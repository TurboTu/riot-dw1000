Destroys a spinlock.

Warning
See the warning in pthread_spinlock_t.

Destroying a spinlock while a thread is waiting for it causes undefined behavior. This is a no-op.

Parameters
[in,out]lockDatum to destroy.
Returns
0 on success. EINVAL if lock == NULL.
