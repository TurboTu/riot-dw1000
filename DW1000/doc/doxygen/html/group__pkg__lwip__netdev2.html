err_t lwip_netdev2_init ( struct netif *  netif)

Initializes the netdev2 adapter.

Should be passed to lwIP's netif_add() with the state parameter parameter of that function set to an existing netdev2_t instance

Precondition
netif->state is set to an existing netdev2_t instance.
Parameters
[in]netifThe network interface intended to be initialized.
Returns
ERR_OK on success.
ERR_IF on error.
