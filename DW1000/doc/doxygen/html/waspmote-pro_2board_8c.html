void led_init ( void  )

Initialize the boards on-board LEDs (green and red)

Initialize the boards on-board LED.

The LED initialization is hard-coded in this function. As the LED is soldered onto the board it is fixed to its CPU pins.

The LEDs are connected to the following pins:

  • LED_GREEN: PC1
  • LED_RED: PD6
