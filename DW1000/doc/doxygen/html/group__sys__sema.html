◆ sema_destroy()

void sema_destroy ( sema_t *  sema)

Destroys a semaphore.

Precondition
(sema != NULL)
See also
The Open Group Base Specifications Issue 7, sem_destroy()

Destroying a semaphore upon which other threads are currently blocked will wake the other threads causing the sema_wait (or sema_wait_timed) to return error (-ECANCELLED).

Parameters
[in]semaThe semaphore to destroy.
