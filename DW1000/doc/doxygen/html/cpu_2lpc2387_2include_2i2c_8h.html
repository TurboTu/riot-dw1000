Initialize the I2C-Interface with the appropriate operation mode, baud rate and other parameter. The user can optionally give a master interrupt handler. If the user does not declare a handler, an appropriate interrupt is automatically registered for the specific i2c interface.

Parameters
[in]i2c_interfacethe i2c interface, several interfaces can be selected: i2c0, i2c1 and i2c2.
[in]i2c_modethe operating mode.
[in]slave_addrthe slave address.
[in]baud_ratethe baud rate.
[in]handlerthe I2C2 Interrupt handler (optional); by giving a NULL as parameter, an appropriate interrupt is automatically selected.
Returns
true if the I2C interrupt handler was installed correctly, otherwise false.
