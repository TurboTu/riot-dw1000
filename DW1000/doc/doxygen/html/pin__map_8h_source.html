214 //
215 // Configures the specified ADC pin to function as an ADC pin.
216 //
217 // \param ulName is one of the valid names for the ADC pins.
218 //
219 // This function takes on of the valid names for an ADC pin and configures
220 // the pin for its ADC functionality depending on the part that is defined.
221 //
222 // The valid names for the pins are as follows: \b ADC0, \b ADC1, \b ADC2,
223 // \b ADC3, \b ADC4, \b ADC5, \b ADC6, or \b ADC7.
224 //
225 // \sa GPIOPinTypeADC() in order to configure multiple ADC pins at once.
226 //
227 // \return None.
228 //
229 //*****************************************************************************
