◆ SPI_NUMOF

#define SPI_NUMOF   1 /* set to 0 to disable SPI */

SPI configuration.

The atmega2560 has only one hardware SPI with fixed pin configuration, so all we can do here, is to enable or disable it...

The fixed pins used, are: MOSI - PB3 (Arduino pin 11) MISO - PB4 (Arduino pin 12) SCK - PB5 (Arduino pin 13) SS - PB2 (Arduino pin 10) -> this pin is configured as output, but not used
