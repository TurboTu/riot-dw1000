Initialize an instance of the NOMAC layer.

The initialization starts a new thread that connects to the given netdev device and starts a link layer event loop.

Parameters
[in]stackstack for the control thread
[in]stacksizesize of stack
[in]prioritypriority for the thread housing the NOMAC instance
[in]namename of the thread housing the NOMAC instance
[in]devnetdev device, needs to be already initialized
Returns
PID of NOMAC thread on success
-EINVAL if creation of thread fails
-ENODEV if dev is invalid
