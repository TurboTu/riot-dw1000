96 /* chunk_size is the number of bytes that will be sent with each */
97 /* "C" (Copy RAM to Flash) command. This must be one of the sizes */
98 /* supported by that command. Beware that different chips support */
99 /* different sets of sizes, so check the user manual specific to */
100 /* the chip. You must choose a chunk_size which is an an integer */
101 /* multiple of all the sector sizes, and it must be able to fit */
102 /* entirely within the RAM (allowing for the bootloader memory and */
103 /* stack usage). Currently, all available chunk sizes meet these */
104 /* requirements, but who knows what Philips will do in the future? */
105 //
