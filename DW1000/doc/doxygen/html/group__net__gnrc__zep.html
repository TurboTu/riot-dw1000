◆ gnrc_zep_init()

kernel_pid_t gnrc_zep_init ( gnrc_zep_t *  dev,
uint16_t  src_port,
ipv6_addr_t *  dst,
uint16_t  dst_port 
)

Initializion of the ZEP thread and device.

Parameters
[in]devNetwork device, will be initialized.
[in]src_portSource port to use in UDP datagrams. Also the port Zigbee Encapsulation Protocol registers to in Network protocol registry.
[in]dstDestination address to use in IPv6 packets.
[in]dst_portDestination port to use in UDP datagrams.
Returns
PID of the ZEP thread on success.
-EADDRINUSE, if src_port is already ready registered to Network protocol registry.
-EEXIST, if ZEP thread was already created.
-EINVAL, if GNRC_ZEP_PRIO is greater than or equal to SCHED_PRIO_LEVELS
-ENODEV, if dev is NULL.
-ENOTSUP, if dst is NULL or unspecified address (::).
-EOVERFLOW, if there are too many threads running already
