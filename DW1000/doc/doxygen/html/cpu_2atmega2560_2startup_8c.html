void init7_ovr ( void  )

This pair of functions hook circumvent the call to main.

avr-libc normally uses the .init9 section for a call to main. This call seems to be not replaceable without hacking inside the library. We circumvent the call to main by using section .init7 to call the function reset_handler which therefore is the real entry point and section .init8 which should never be reached but just in case jumps to exit. This way there should be no way to call main directly.
