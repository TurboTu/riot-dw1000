Function: dwt_getrangebias()

Description: This function is used to return the range bias correction need for TWR with DW1000 units.

input parameters:

Parameters
chan- specifies the operating channel (e.g. 1, 2, 3, 4, 5, 6 or 7)
range- the calculated distance before correction
prf- this is the PRF e.g. DWT_PRF_16M or DWT_PRF_64M

output parameters

returns correction needed in meters
