1 Hello World!
2 ============
3 
4 This is a basic example how to use RIOT in your embedded application.
5 It prints out the famous text `Hello World!`.
6 
7 This example should foremost give you an overview how to use the Makefile system:
8 
9 * First you must give your application a name, which is commonly the same as the name of the directory it resides in.
10  Then you can define a default BOARD for which the application was written.
11  By using e.g. `make BOARD=msba2` you can override the default board.
12  With `make buildtest` the application gets compiled for all supported boards.
13 
14 * The variable `RIOTBASE` contains an absolute or relative path to the directory where you have checked out RIOT.
15  If your code resides in a subdirectory of RIOT, then you can use `$(CURDIR)` as it's done in here.
16 
17 * The variable `QUIET`, which is either `1` or `0`, defines whether to print verbose compile information, or hide them, respectively.
18 
19 * The last line of your Makefile must be `include $(RIOTBASE)/Makefile.include`.
20 
21 The code itself may look like your usual *C* beginners hello-world example.
