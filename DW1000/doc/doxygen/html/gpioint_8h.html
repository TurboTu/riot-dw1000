GPIO IRQ handler setup.

Parameters
[in]portCPU specific port number (starting at 0)
[in]bitmaskOne or more bits for which to set the handler
[in]flagsA combination of GPIOINT_RISING_EDGE and GPIOINT_FALLING_EDGE
[in]callbackA pointer to a handler function
Return values
truesuccessful
falsefailed

To enable interrupt handling for a pin flags and callback must be non-zero. To disable interrupt handling flags and callback shall be zero.
