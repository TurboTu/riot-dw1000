#define AVR_CONTEXT_SWAP_INIT
Value:
do { \
DDRJ |= (1 << PJ6); \
PCICR |= (1 << PCIE1); \
PCMSK1 |= (1 << PCINT15); \
} while (0)

Context swap defines Setup to use PJ6 which is pin change interrupt 15 (PCINT15) This emulates a software triggered interrupt
