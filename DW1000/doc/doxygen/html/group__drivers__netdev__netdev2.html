Detailed Description

This is the second version of a generic low-level network driver interface.

This interface is supposed to be a low-level interface for network drivers.

Example call flow:

  1. packet arrives for device
  2. The driver previously registered an ISR for handling received packets. This ISR then calls netdev->event_callback() with event := NETDEV2_EVENT_ISR (from Interrupt Service Routine) which wakes up event handler
  3. event handler calls netdev2->driver->isr() (from thread context)
  4. netdev->driver->isr() calls netdev->event_callback() with event := NETDEV2_EVENT_RX_COMPLETE
  5. netdev->event_callback() uses netdev2->driver->recv() to fetch packet
