Detailed Description

Low-level RTT interface implementation for Freescale Kinetis MCUs. Freescale's RTC module is what RIOT calls a Real-Time Timer (RTT), a simple counter which counts seconds; RIOT Real- Time Clocks (RTC) counts seconds, minutes, hours etc. We provide an RTT->RTC wrapper layer in a separate file to allow using the RTT as a system real time clock.
