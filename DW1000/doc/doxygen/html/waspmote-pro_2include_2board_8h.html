#define AVR_CONTEXT_SWAP_INIT
Value:
do { \

PCICR |= (1 << PCIE0); \
PCMSK0 |= (1 << PCINT5); \
} while (0)

Context swap defines Setup to use PB5 which is pin change interrupt 5 This emulates a software triggered interrupt
